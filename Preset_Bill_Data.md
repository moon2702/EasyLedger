- **微信**：200 - 23 - 55 + 500 = 622元
- **支付宝**：300 - 89 + 300 - 200 = 311元

- **总资产**：319 + 19700 + 622 + 311 = 20952元
- **总负债**：99元（花呗欠款）

4. **债务管理**：信用卡和花呗的还款
